# SOM Clustering Analysis for Cotton Project

This repository contains an R script (`SOM_clustering_analysis.R`) that performs:
- Self‚ÄêOrganizing Map (SOM) clustering
- Outlier removal
- Profile interpolation
- Generating various output files (CSV, PDF plots) for cotton protein fraction data.

## Requirements
- R version 4.1 or higher
- Packages: `kohonen`, `tempR`, `dplyr`, `ggplot2`, `openxlsx`, `tibble`, `tidyverse`, `Rcpp`

## Usage
1. Clone the repo: `git clone https://github.com/<YourUserName>/SOM_Clustering_CottonPJ`
2. Open `SOM_clustering_analysis.R` in R / RStudio.
3. Adjust the `workingDir` paths to point to your data folder.
4. Run the script. It will generate output CSV and PDF files in the specified folders.
